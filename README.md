# 🧠 React Quiz App

A simple and interactive quiz app built with **React.js** to test your knowledge on fun topics like superheroes and more.

## 🚀 Features

- Dynamic quiz questions loaded from a data file
- Interactive UI with option selection
- Score tracking
- Responsive design
- Built with Vite for fast development

## 🧩 How to Run

git clone https://github.com/MAHERAOUNALLAH/quiz-app.git
cd quiz-app

Install dependencies

npm install

Run the app

npm run dev
